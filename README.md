# Swasthya-AI

A real-time multilingual healthcare voice assistant powered by Murf Falcon TTS.

## Vision
To build a low-latency voice-based medical triage assistant capable of handling health-related queries in natural conversation.

## Tech Stack (Planned)
- Frontend: HTML, JavaScript
- Backend: Python (FastAPI)
- Text-to-Speech: Murf Falcon
- LLM: TBD
